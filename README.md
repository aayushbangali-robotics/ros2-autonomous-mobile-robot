# ROS2 Autonomous Mobile Robot

# Overview
This project focuses on building an autonomous mobile robot using ROS2 for indoor navigation in unknown environments.

# Planned Features
- Differential drive robot
- LiDAR-based SLAM
- Autonomous navigation using Nav2
- Obstacle avoidance

# Tech Stack
- ROS2 Humble
- Gazebo
- Python / C++
